Project MoMA Atlantic Pacific

We created a program that would over a six week period, tease an installation in the Brooklyn Atlantic Pacific Station, then delight commuters with an engaging interactive display of over 50 works from MoMA's permanent collection.

This was my first high-profile project that would go on the receive both high traffic and acclaim. This was in 2008 when the browser was a much different thing than it is today. I was faced with the challenge of creating a single-page web application that would animate between screens as if the user was looking out the window of the subway.
